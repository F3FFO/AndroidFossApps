## About

This repository maintains a curated list of **1000+** free and open source Android apps, organized by category.

- **Interactive website** — Search, filter by category, toggle dark mode
- **[APPS.md](APPS.md)** — Full markdown list with all apps and categories

## Abbreviations

| Abbreviation | Meaning |
| :----------: | ------- |
|  **`DEAD`**  | App whose development is discontinued/paused indefinitely. |
|  **`FORK`**  | Project that uses the source code of another project as a starting point, then adds modifications on top of it. |
|  :warning:   | Only the application is FOSS. The service is based on a closed-source or non-FOSS system. |

## Contributing

Want to add an app? Edit `apps.json` and submit a pull request. The `APPS.md` file is auto-generated from `apps.json` — please don't edit it directly.

To add a recently-added entry, also update `recently-added.json`.
